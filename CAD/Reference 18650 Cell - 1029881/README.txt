This is a 18650 cell reference model I put together for a project and thought it might be useful to someone so I uploaded it here. The model is accurate to a Samsung 18650 cell I measured in the real world so the model should be more or less 100% accurate to a 18650 cell and is well detailed to be just like the real thing.
